# Nbots

- ### 네이버 블로그, 카페 등의 플랫폼에서 자동적으로 번거로운 작업들을 대신 해주는 플랫폼

- ### JSP 백엔드에서 네이버 API에 직접 접근하여 정보를 얻어오고 작업을 수행합니다.

- ### 2021년에 마지막으로 수정된 소스코드이므로 작동되지 않는 기능이 있을 수 있습니다.
